Welcome to my blog. Posts in this repo are loaded at runtime on [vatsalverma.in/blog](https://www.vatsalverma.in/blog).

## Getting started

Add a new `.md` file under `blog/` with YAML frontmatter. The site picks it up automatically — no portfolio redeploy needed.

```bash
# Example: cp MD_TEMPLATE.md blog/my-post.md
```

Happy writing.
